## 四、遇到的问题及解决方案

| 序号 | 问题 | 原因 | 解决方案 |
|------|------|------|----------|
| 1 | 后端启动日志写权限失败 | 沙箱 `/home/waithooo/logs/` 只读 | 改 `application-local.yaml` logging path 为 `/tmp/` |
| 2 | 菜单中文显示为乱码 "å'ç¥¨ç®¡ç†" | 通过 `docker exec -i` 管道执行 SQL 时 MySQL 客户端默认 latin1 编码导致二次编码 | 删除后加 `--default-character-set=utf8mb4` 重新执行 |
| 3 | API 返回 404 | 实际是 `/admin-api/` 前缀缺失 | curl 测试时补全前缀，Controller 本身正确 |
| 4 | vue-tsc OOM | 沙箱内存不足 (2GB) | 改用 Vite 运行时编译验证，全部 HTTP 200 |
