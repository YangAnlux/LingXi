# 密讯 (mitedtsm) 项目需求分析报告

## 1. 项目概述

### 1.1 项目背景

**密讯 (mitedtsm)** 是一款以开发者为中心的企业级快速开发平台，基于 RuoYi-Vue Pro 重构优化而来。项目采用 MIT 开源协议，个人与企业可 100% 免费使用。

### 1.2 项目定位

打造中国第一流的快速开发平台，提供完整的企业级解决方案，支持快速构建各类业务系统。

### 1.3 核心价值

| 价值点 | 描述 |
|--------|------|
| 全开源 | 所有代码全部开源，无商业版本限制 |
| 高扩展性 | 模块化架构，按需加载业务模块 |
| 多租户 | 支持 SaaS 场景下的多租户部署 |
| 高效率 | 代码生成器一键生成前后端代码 |
| 多端适配 | 支持 PC、H5、小程序、APP 等多终端 |

## 6. 数据需求

### 6.1 数据库支持

| 数据库 | 类型 | 用途 |
|--------|------|------|

| Redis | 缓存数据库 | 缓存、会话、分布式锁 |
| PostgreSQL | 关系型数据库 | 可选数据库方案 |

| SQL Server | 关系型数据库 | 可选数据库方案 |
| 达梦 DM | 国产数据库 | 信创场景支持 |

### 6.2 数据结构

系统采用 MyBatis Plus 作为 ORM 框架，数据结构遵循以下规范：

- 表名：`t_模块名_业务名`（如 `t_system_user`）
- 字段命名：小驼峰命名法
- 主键策略：雪花算法生成的分布式 ID
- 逻辑删除：支持软删除机制
- 租户字段：支持多租户隔离

### 10.1 项目特点

1. **模块化架构**：12 个业务模块可独立部署，按需加载
